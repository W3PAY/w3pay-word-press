Accept payments via crypto on your store using W3PAY.

== Description ==

Accept BUSD, USDT, USDC, DAI and more directly on your store with the W3PAY Web3 Payment System for WooCommerce. The buyer can pay with any token. The seller receives in the desired USD stablecoin. Documents and verification are not required.

= Accept crypto payments easily and directly on your store =

The w3pay plugin extends WooCommerce allowing you to take payments directly on your store.

Available for multiple blockchain networks and tokens:

* Smart Chain - Testnet (Tokens: BUSD, USDT, DAI, ETH, BNB)
* Binance Smart Chain Mainnet (Tokens: BUSD, USDT, DAI, USDC, BTCB, ETH, BNB)
* Polygon Mainnet (Tokens: USDT, USDC, DAI, WBTC, ETH, MATIC)
* Avalanche C-Chain (Tokens: USDT, USDC, DAI, BTC.b, WETH.e, AVAX)
* Fantom Opera (Tokens: fUSDT, USDT, DAI, WBTC, WETH, FTM)

= Why choose w3pay? =

Documents and verification are not required to use the method. Available to every person. The buyer pays on your site using Trust Wallet or MetaMask wallets. The buyer can pay with any token. The seller receives in stablecoin. The buyer receives 1% cashback. The seller receives 1% less. W3PAY has no setup fees, no monthly fees, no hidden costs!

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of the WooCommerce w3pay plugin, log in to your WordPress dashboard, navigate to the Plugins menu and click "Add New".

In the search field type "w3pay" and click Search Plugins. Once you've found our plugin you can view details about it such as the point release, rating, and description. Most importantly, of course, you can install it by simply clicking "Install Now", then "Activate".

= Manual installation =

Download zip archive from [GitHub W3PAY](https://w3pay.github.io/plugins/w3payWordPress/w3pay.zip). To do an manual installation method of the WooCommerce w3pay plugin, log in to your WordPress dashboard, navigate to the Plugins menu and click "Add New" and click "Upload plugin".  The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
